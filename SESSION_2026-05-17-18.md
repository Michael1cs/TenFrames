### Romanian grammar fix
ElevenLabs reads bare digits as Italian/French numerals on some clips
("doi rachete", "cinquo rachete", "ai zeci comete"), and it reads "2" as
masculine "doi" regardless of noun gender.

- `/tmp/ro_grammar_fix.mjs` rewrote 1181 RO entries so digits become
  gender-correct words. Examples:
  - `have_rocket_2`: "Ai 2 rachete." → "Ai două rachete."
  - `have_fish_2`: "Ai 2 pești." → "Ai doi pești."
  - `pre_have_space_10`: "Ai 10 rachete." → "Ai zece rachete."
- Gender map for all 32 noun keys + 9 theme keys is inline in the
  script (rocket=f, fish=m, trophy=n, …).
- 1176 RO clips regenerated. EN/DE library stayed unchanged.

### Module-level voice queue
- `useVoice` used to maintain a queue per instance, which let
  Adventure narration overlap GameShell's reward/praise chain (each
  had its own queue).
- Queue is now a single FIFO at module scope. Every `play`,
  `playRandom`, and `playSequence` from any screen enqueues with a
  250 ms gap between clips. `stop()` clears the queue and the
  in-flight clip.
- The local `voiceQueueRef`/`voiceQueueBusyRef` in `GameShell` were
  redundant — removed. `voice.stop()` on mode change is enough to
  reset everything.

## Voice generation cheat sheet

```bash
# Single language, full regen (uses pinned voice + model):
ELEVENLABS_API_KEY=… node scripts/gen-voice.mjs --lang=ro

# Specific IDs only:
ELEVENLABS_API_KEY=… node scripts/gen-voice.mjs --ids=welcome,instr_make_ten

# Override the voice/model for an experiment:
ELEVENLABS_API_KEY=… VOICE_ID_EN=XYZ ELEVENLABS_MODEL_EN=eleven_v3 \
  node scripts/gen-voice.mjs --ids=welcome --lang=en --force
```

After generating, the Android raw dir is synced manually:
```bash
cp assets/audio/voice_<lang>_*.mp3 \
  android/app/src/main/res/raw/
```

iOS picks new files up automatically on rebuild (filenames are stable;
contents change in-place). New filenames need a `pbxproj` entry —
adapt `/tmp/patch_pbxproj_new.mjs`.
